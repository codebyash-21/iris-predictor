# 🌸 Iris Species Classifier (Supervised Learning)

This project is a complete end-to-end Machine Learning application that classifies Iris flowers into three species based on physical measurements.

## 🚀 Key Features
* **Model:** Random Forest Classifier (Supervised Learning).
* **UI:** Clean, interactive web interface built with Streamlit.
* **Accuracy:** High performance on the classic Iris Dataset.

## 🛠️ How to Run Locally
1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `streamlit run app.py`

## 📊 Dataset Features
The model analyzes four key features:
1. Sepal Length
2. Sepal Width
3. Petal Length
4. Petal Width
